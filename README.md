# Supermarket Management System

A simple C++ console-based supermarket management system that allows users to input their name, ID, and password. The system uses object-oriented principles to manage user data.

## Files Included
- `main.cpp` — Entry point of the program
- `Supermarket.cpp` — Implementation of supermarket logic
- `Supermarket.h` — Header file for supermarket class
- `Supermarket.slnx` — Visual Studio solution filter (open this or the `.vcxproj` manually)

## How to Run (with visual studio)
1. Clone the repository
2. Open `Supermarket.slnx` in Visual Studio  
   _or_ open `SupermarketManagementSystem.vcxproj` directly
3. Build the project
4. Run with `Ctrl + F5`
